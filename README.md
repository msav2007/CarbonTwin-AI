# CarbonTwin AI

AI-powered carbon intelligence twin for PromptWars Challenge 3.

## Quick Start

```bash
cp .env.example .env.local
# Fill in Supabase + Gemini keys
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## MVP Features

1. Landing Page
2. Authentication (Supabase)
3. 5-Step Onboarding
4. Carbon Twin Generation
5. Dashboard
6. Future Simulator
7. What-If Time Machine
8. AI Receipt Analysis
9. Twin Speaks
10. Carbon Budget Meter

See [docs/MVP_PLAN.md](docs/MVP_PLAN.md) for architecture, roadmap, and deployment.

## Stack

- Next.js 14 · TypeScript · Tailwind · ShadCN
- Supabase (Auth + Postgres)
- Google Gemini 1.5 Flash
- Vercel
